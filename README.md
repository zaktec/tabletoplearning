# Tabletoplearning

Tabletoplearning is a modern educational landing website built with Astro. It showcases subjects, course options, support levels, and a polished learning-focused brand experience.

## What this project includes

- A warm, modern landing page for education and tutoring
- Subject-based course cards that link to dedicated course pages
- Support tier sections for Bronze, Silver, and Gold plans
- A simple, brand-consistent layout for future expansion

## Project documentation

- Branding guide: [docs/branding.md](docs/branding.md)
- Project overview: [docs/project-overview.md](docs/project-overview.md)

## Run locally

1. Install dependencies
   ```bash
   npm install
   ```
2. Start the development server
   ```bash
   npm run dev
   ```
3. Open the local site in your browser

## Build

```bash
npm run build
```

## Notes

This project is currently a static Astro site and can be extended with authentication, a CMS, or a larger course catalog later.
